# Issue Convention

## 제목 형식

```text
[Init] {작업명}
```

## 예시

```text
[Init] 리액트 초기세팅
[Init] Tanstack Query 세팅
[Init] color, font 시스템 세팅 및 global css 세팅
```

## 본문 구성

Issue 본문에는 아래 내용을 포함합니다.

```text
목적
작업 목록
완료 기준
참고 자료
```

## 작성 규칙

- 초기세팅 작업 issue는 `[Init]` prefix를 사용합니다.
- issue는 한 step의 목표와 완료 기준을 설명합니다.
- reference가 있다면 issue 본문에 경로를 함께 기록합니다.
- 실제 secret 값은 issue에 작성하지 않습니다.
